# Implementation plan

## Integration contract

**Entrypoint:** `linkshelf/cmd/server/main.go`

**Startup / dependency order (SPEC + architecture):**

- Open SQLite file `linkshelf.db` in the current working directory.
- `InitSchema(db)` then `store.DB = db`.
- Register handlers and static routes.
- `http.ListenAndServe(":8080", nil)` and log `listening on :8080`.
- `cd linkshelf && go test ./...` — green (packages without `*_test.go` still compile).
- `cd linkshelf && go run ./cmd/server` — UI loads; add one link, see it, delete it.

**HTTP routes (SPEC verbatim):**

| Method | Path |
|--------|------|
| GET | `/` |
| GET | `/static/{file}` |
| GET | `/api/links` |
| POST | `/api/links` |
| DELETE | `/api/links/{id}` |

**Per-file exports:** see `architecture.md` per-file ownership table; use only names documented there.
